**РӣРөР»РөРәР°** вҖ” СҶРө РІРөРұ-Р·Р°СҒСӮРҫСҒСғРҪРҫРә РҙР»СҸ РІС–РҙСҒСӮРөР¶РөРҪРҪСҸ РІР°РіС–СӮРҪРҫСҒСӮС–, СҸРәРёР№ РҙРҫРҝРҫРјР°РіР°С”
РәРҫСҖРёСҒСӮСғРІР°СҮР°Рј СҒРҝРҫСҒСӮРөСҖС–РіР°СӮРё Р·Р° СҒРІРҫС—Рј СҒСӮР°РҪРҫРј, РөРјРҫСҶС–СҸРјРё СӮР° СҖРҫР·РІРёСӮРәРҫРј РҙРёСӮРёРҪРё СүРҫСӮРёР¶РҪСҸ.
РўСғСӮ РҝСҖРөРҙСҒСӮР°РІР»РөРҪР° frontend СҮР°СҒСӮРёРҪР° Р·Р°СҒСӮРҫСҒСғРҪРәСғ **РӣРөР»РөРәР°**, СҒСӮРІРҫСҖРөРҪР° СҒСӮСғРҙРөРҪСӮР°РјРё
РәСғСҖСҒСғ FullStack Сғ СҲРәРҫР»С– GoIT РҙР»СҸ РҝСҖР°РәСӮРёРәРё Next.js СӮР° РәРҫРјР°РҪРҙРҪРҫС— СҖРҫРұРҫСӮРё Р·
Git/GitHub.

## РһСҒРҪРҫРІРҪРёР№ С„СғРҪРәСҶС–РҫРҪР°Р»

- РҗСғСӮРөРҪСӮРёС„С–РәР°СҶС–СҸ РәРҫСҖРёСҒСӮСғРІР°СҮС–РІ
- РЈРҝСҖР°РІР»С–РҪРҪСҸ РҝСҖРҫС„С–Р»РөРј
- Р’С–РҙСҒСӮРөР¶РөРҪРҪСҸ РҝРҫСӮРҫСҮРҪРҫРіРҫ СӮРёР¶РҪСҸ РІР°РіС–СӮРҪРҫСҒСӮС–
- CRUD (Create/Read/Update/Delete) РҙР»СҸ РҪРҫСӮР°СӮРҫРә Р· РІР°Р¶Р»РёРІРёРјРё СҒРҝСҖР°РІР°РјРё
- РқРҫСӮР°СӮРәРё РҝСҖРҫ СҒР°РјРҫРҝРҫСҮСғСӮСӮСҸ
- РҹР»Р°РҪСғРІР°РҪРҪСҸ (Р·Р°РјС–СӮРәРё, СҶС–Р»С–)
- РӣРҫРі РҪР°СҒСӮСҖРҫСҺ
- Р©РҫРҙРөРҪРҪС– РҝРҫСҖР°РҙРё РҙР»СҸ РјР°РјРё
- Р—РјС–РҪР° РәРҫР»СҢРҫСҖСғ СӮРөРјРё РІ Р·Р°Р»РөР¶РҪРҫСҒСӮС– РІС–Рҙ СҒСӮР°СӮС– РјР°Р»СҺРәР°
- РҶРҪС„РҫСҖРјР°СҶС–СҸ РҝСҖРҫ СҖРҫР·РјС–СҖ РҙРёСӮРёРҪРё РҪР° РәРҫР¶РҪРҫРјСғ РөСӮР°РҝС–

## Р—Р°РҝСғСҒРә РҝСҖРҫС”РәСӮСғ Р»РҫРәР°Р»СҢРҪРҫ

```bash
# РҡР»РҫРҪСғРІР°СӮРё СҖРөРҝРҫР·РёСӮРҫСҖС–Р№
git clone https://github.com/Oleksii996/project-decodery-front.git

# РҹРөСҖРөР№СӮРё РІ РҝР°РҝРәСғ
cd project-decodery-front

# Р’СҒСӮР°РҪРҫРІРёСӮРё Р·Р°Р»РөР¶РҪРҫСҒСӮС–
npm install

# Р—Р°РҝСғСҒСӮРёСӮРё dev СҒРөСҖРІРөСҖ
npm run dev
```
